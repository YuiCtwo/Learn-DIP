# Learn-DIP

《数字图像处理第四版》书中等一些基础算法的实现，包括相关库的调用

大三下选修了数字图像处理，书上讲的内容有很多，也有不少有意思的并且经典的算法，碍于学的时候的时间原因，没有整理实验课写的代码，我计划着大三暑假，大四整理整理并补充一些书本外的经典 DIP 用到的一些算法。

- 记录一些书上算法的复现和一些平时用到的算法，如果有库尽量会使用库函数来更快捷的完成，毕竟重复的轮子尽量少造。
